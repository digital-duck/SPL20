def greeting(input, lang):
    """
    Generates a personalized greeting message.

    Args:
        input (str): The person's name.
        lang (str): The language of the greeting.

    Returns:
        str: A personalized greeting message.
    """
    
    # Predefined greetings in different languages
    greetings = {
        "zh": f"你好，{input}！",
        "en": f"Hello, {input}!",
        "fr": f"Bonjour, {input}!",
        "es": f"Hola, {input}!"
    }
    
    # Check if the language is supported
    if lang in greetings:
        return greetings[lang]
    else:
        return "Sorry, we don't support this language."

# Example usage:
print(greeting("Wen-Guang", "zh"))  # Output: 你好，Wen-Guang！
```

This function takes two parameters: the person's name and their preferred language. It returns a personalized greeting message in the specified language. If the language is not supported, it returns an error message.
